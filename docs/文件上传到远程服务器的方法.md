# 文件上传到远程 Windows 服务器的方法

## 方法1：远程桌面直接复制粘贴（最简单）⭐推荐

### 步骤：

1. **启用远程桌面剪贴板共享**
   - 打开远程桌面连接：`Win + R` → 输入 `mstsc` → 回车
   - 点击"显示选项"
   - 在"本地资源"标签页中，勾选"剪贴板"

2. **连接服务器**
   - 计算机名：`119.91.36.69`
   - 用户名：`Administrator`
   - 密码：`aA1234567890`
   - 点击"连接"

3. **直接复制粘贴文件**
   - 在本地，找到 `dist\LabClassificationService.exe`
   - 右键复制（Ctrl+C）
   - 切换到远程桌面窗口
   - 在远程桌面的某个文件夹（如 `C:\temp` 或桌面）中
   - 右键粘贴（Ctrl+V）

**优点**：简单直接，无需额外工具  
**缺点**：文件较大时可能较慢

## 方法2：使用 PowerShell Remoting（适合程序员）

### 步骤：

1. **在本地 PowerShell 中执行**（以管理员身份运行）

```powershell
# 创建远程会话
$credential = Get-Credential
# 用户名输入：Administrator
# 密码输入：aA1234567890

$session = New-PSSession -ComputerName 119.91.36.69 -Credential $credential

# 创建临时目录（如果不存在）
Invoke-Command -Session $session -ScriptBlock {
    if (!(Test-Path "C:\temp")) {
        New-Item -ItemType Directory -Path "C:\temp"
    }
}

# 复制文件到服务器
Copy-Item -Path "dist\LabClassificationService.exe" -Destination "C:\temp\LabClassificationService.exe" -ToSession $session

# 断开连接
Remove-PSSession $session
```

**优点**：自动化，适合批量文件  
**缺点**：需要 PowerShell 和网络配置允许

## 方法3：使用 FTP/SFTP 工具（FileZilla）

### 步骤：

1. **下载 FileZilla**（如果需要）：https://filezilla-project.org/

2. **配置 FTP 服务器**（如果服务器已启用 FTP）
   - 主机：`119.91.36.69`
   - 用户名：`Administrator`
   - 密码：`aA1234567890`
   - 端口：21（FTP）或 22（SFTP）

3. **上传文件**
   - 连接后，左侧是本地文件，右侧是远程文件
   - 拖拽 `dist\LabClassificationService.exe` 到远程目录

**注意**：需要服务器启用 FTP/SFTP 服务

---

## 方法4：使用 Windows 网络共享（SMB）

### 步骤：

1. **在服务器上创建共享文件夹**（通过远程桌面）

2. **在本地映射网络驱动器**
   - `Win + R` → 输入：`\\119.91.36.69\C$` → 回车
   - 输入用户名和密码
   - 可以将文件直接复制到共享文件夹

**优点**：类似本地文件操作  
**缺点**：需要配置网络共享

---

## 方法5：使用 WinSCP（推荐用于大文件）⭐

### 步骤：

1. **下载 WinSCP**：https://winscp.net/

2. **配置连接**
   - 文件协议：选择 "SFTP" 或 "FTP"
   - 主机名：`119.91.36.69`
   - 端口号：22（SFTP）或 21（FTP）
   - 用户名：`Administrator`
   - 密码：`aA1234567890`

3. **连接并上传**
   - 点击"登录"
   - 左侧选择本地文件 `dist\LabClassificationService.exe`
   - 右侧选择远程目录（如 `/C/temp/` 或桌面）
   - 拖拽或使用"上传"按钮

**优点**：界面友好，支持断点续传  
**缺点**：需要安装工具，需要服务器开启 FTP/SFTP

---

## 最简单的方法（推荐新手）

**使用远程桌面的复制粘贴功能：**

1. 打开远程桌面连接（`mstsc`）
2. 在"本地资源"中勾选"剪贴板"
3. 连接服务器
4. 在本地复制文件（Ctrl+C）
5. 在远程桌面中粘贴（Ctrl+V）

## 快速验证上传的文件

上传后，在远程桌面的 PowerShell 或 CMD 中：

```cmd
cd C:\temp
dir LabClassificationService.exe
```

如果显示文件信息，说明上传成功！
